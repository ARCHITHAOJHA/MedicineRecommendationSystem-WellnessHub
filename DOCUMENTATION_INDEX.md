## 🚀 Quick Start Guide

### 1. Run the Application
```bash
python main.py
```

### 2. Open Browser
```
http://127.0.0.1:5000
```

### 3. Use Speech Recognition
1. Click "Start Speech Recognition"
2. Speak your symptoms (e.g., "fever, cough")
3. Text auto-fills the input field
4. Click "Predict"
5. Get recommendations
